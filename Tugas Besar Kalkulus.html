Akar-akar Persamaan Kuadrat, Pengertian, Jenis, dan Penyelesaiannya

A. Pengertian Akar-Akar Persamaan Kuadrat

Dalam matematika, persamaan kuadrat adalah persamaan polinomial yang paling sederhana dalam bentuk ax^2 + bx + c = 0, dengan a, b, dan c sebagai koefisien bilangan real dan a ≠ 0.

Setiap persamaan memiliki dua kemungkinan nilai x karena sifatnya yang merupakan persamaan dengan pangkat tertinggi dua.

Berdasarkan penjelasan tersebut, akar-akar persamaan kuadrat adalah nilai pengganti x yang memenuhi persamaan kuadrat yang biasanya disimbolkan sebagai x1 dan x2.

B. Jenis Akar-akar Persamaan Kuadrat

Secara umum, persamaan kuadrat dibagi menjadi empat, yaitu sebagai berikut.

1. Persamaan Kuadrat Biasa

• Persamaan kuadrat biasa adalah persamaan kuadrat yang nilai a = 1.

• Berikut ini contohnya. x2 + 3x + 2 = 0

2. Persamaan Kuadrat Murni

• Persamaan kuadrat murni adalah persamaan kuadrat yang nilai b = 0.

• Berikut ini contohnya. x2 + 2 = 0

3. Persamaan Kuadrat Tak Lengkap

• Persamaan kuadrat tak lengkap adalah persamaan kuadrat yang nilai c = 0.

• Berikut ini contohnya. x2 + 3x = 0

4. Persamaan Kuadrat Rasional

•Persamaan kuadrat rasional adalah persamaan kuadrat yang nilai koefisien dan konstantanya berupa bilangan rasional.

•Berikut ini contohnya. 4x2 + 3x + 2 = 0

Cara Menyelesaikan Persamaan Kuadrat:

Untuk menyelesaikan persamaan kuadrat, kita dapat menggunakan rumus kuadratik yang dikenal sebagai rumus abc:

Jika ax^2 + bx + c = 0, maka akar-akarnya dapat dihitung menggunakan rumus:

        x = (-b ± √(b2 - 4ac)) / 2a

Dimana:

  • a, b, dan c adalah koefisien dari persamaan kuadrat
  • √ adalah simbol untuk akar kuadrat
  • b2 - 4ac adalah diskriminan

Persamaan kuadrat dapat memiliki tiga kondisi hasil akar:

  1. Jika diskriminan > 0, maka persamaan kuadrat memiliki dua akar riil yang berbeda.
  2. Jika diskriminan = 0, maka persamaan kuadrat memiliki satu akar riil.
  3. Jika diskriminan < 0, maka persamaan kuadrat memiliki dua akar imajiner.

Contoh Penerapan Rumus Kuadratik:

Misalnya kita memiliki persamaan \(2x^2 - 5x + 2 = 0\). Dalam hal ini, a = 2, b = -5, dan c = 2. Dengan menggunakan rumus kuadratik, kita dapat menghitung nilai x.
